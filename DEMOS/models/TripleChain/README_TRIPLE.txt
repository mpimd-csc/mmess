Matlab function for triple chain mass spring damper model following the model in 

 
N.Truhar and K.Veselic
An efficient method for estimating the optimal dampers' viscosity for
linear vibrating systems using Lyapunov equations
SIAM J. Matriax Anal. Appl. vol.31 no.1 pp 18-39

See test.m for a usage example. 

The model can be scaled to almost arbitrary size and use any number of input and outputs.
